MACHINE LEARNING

1. What is supervised learning in machine learning?

Solution and Explanation:
Supervised learning is a type of machine learning where a model learns from labeled training data. The algorithm maps input data to the correct output based on example data.

Key Features:
  1. Labeled Data: Requires input-output pairs for training.
  2. Training and Testing: Splits data into training and testing sets.
  3. Regression & Classification: Used for both regression and classification tasks.




2. Which algorithm is commonly used for classification problems?

Solution and Explanation:
Classification algorithms are used to categorize data points. Common algorithms include Decision Trees, Support Vector Machines (SVM), and Logistic Regression.

Popular Classification Algorithms:
  1. Decision Trees: Uses a tree-like structure for decision making.
  2. Support Vector Machine (SVM): Finds a hyperplane to separate classes.
  3. Logistic Regression: Estimates probabilities for classification tasks.




3. What is overfitting in machine learning?

Solution and Explanation:
Overfitting occurs when a model learns the training data too well, including noise and irrelevant details, reducing its performance on new data.

Ways to prevent overfitting:
  1. Regularization: Techniques like L1/L2 regularization help control complexity.
  2. Cross-validation: Helps ensure the model generalizes well.
  3. Dropout: Randomly drops neurons during training to prevent dependency on specific features.




4. What is the purpose of the activation function in a neural network?

Solution and Explanation:
The activation function introduces non-linearity into the neural network, allowing it to learn and model complex patterns.

Common Activation Functions:
  1. ReLU: Helps prevent vanishing gradients.
  2. Sigmoid: Used for binary classification tasks.
  3. Softmax: Used in multi-class classification problems.




5. What is the difference between batch gradient descent and stochastic gradient descent?

Solution and Explanation:
Gradient descent is an optimization algorithm used to minimize loss functions. Batch gradient descent updates weights using the entire dataset, while stochastic gradient descent (SGD) updates weights after each training example.

Key Differences:
  1. Batch Gradient Descent: More stable but slower.
  2. Stochastic Gradient Descent: Faster updates but higher variance.
  3. Mini-batch Gradient Descent: A compromise between the two.




6. What is the role of a loss function in machine learning?

Solution and Explanation:
A loss function measures how well the model's predictions match the actual values. It helps guide optimization algorithms like gradient descent.

Types of Loss Functions:
  1. Mean Squared Error (MSE): Used for regression tasks.
  2. Cross-Entropy Loss: Used for classification problems.
  3. Hinge Loss: Common in SVMs.
