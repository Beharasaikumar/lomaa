20. What is the difference between normalization and standardization?

Solution and Explanation:
Normalization scales data between 0 and 1, while standardization centers data around the mean with unit variance.

Key Points:
  • Normalization scales features to a specific range, typically [0, 1].
  • Standardization transforms data to have a mean of 0 and a standard deviation of 1.

24. What type of feature is "Male/Female"?

Solution and Explanation:
"Male/Female" is a categorical feature because it represents different categories (male and female) without any intrinsic order.

Key Points:
  • Categorical features represent discrete values or categories.

25. What is a p-value in hypothesis testing?

Solution and Explanation:
A p-value measures the evidence against the null hypothesis. A lower p-value indicates stronger evidence against the null hypothesis.

Key Points:
  • A p-value below a significance level (e.g., 0.05) suggests that the null hypothesis can be rejected.
  • If the p-value is high, it suggests that the data is consistent with the null hypothesis.




26. What is a Type I error?

Solution and Explanation:
A Type I error occurs when the null hypothesis is rejected when it is actually true (false positive).

Key Points:
  • Type I error is also known as a false positive.
  • It occurs when we incorrectly conclude that there is an effect or relationship when there isn't one.

A normal distribution is typically shaped like a bell curve, where most of the values are clustered around the mean.

Key Points:
  • A normal distribution is symmetric about the mean.
  • It is often used in statistical analysis and hypothesis testing.

Key Points:

30. What is the main goal of Machine Learning?

Solution and Explanation:
The main goal of machine learning is to make predictions based on data, using algorithms to learn from the data.

Key Points:
  • Machine learning models aim to generalize from the data to make accurate predictions.
  • It can be applied to a wide range of tasks, from classification to regression.




31. What is the key difference between regression and classification?

Solution and Explanation:
Regression is used to predict continuous values (e.g., house prices), while classification is used to predict discrete labels (e.g., spam vs. non-spam).

Key Points:
  • Regression: Output is continuous (e.g., 10.5, 100).
  • Classification: Output is discrete (e.g., spam, not spam).




32. Which of the following is an example of a regression problem?

Solution and Explanation:
Predicting house prices is an example of a regression problem because the output is a continuous value (the price).

Key Points:
  • Regression problems deal with continuous data.
  • Classification problems involve discrete labels.




33. What is overfitting in Machine Learning?

Solution and Explanation:
Overfitting occurs when a model learns too much from the training data, including noise, and performs poorly on new, unseen data.

Key Points:
  • Overfitting leads to poor generalization.
  • It can be mitigated by using techniques like cross-validation and regularization.




34. What is clustering in Machine Learning?

Solution and Explanation:
Clustering is the process of grouping similar data points together into clusters without predefined labels.

Key Points:
  • Clustering is an unsupervised learning technique.
  • It helps find patterns and structures in data.




35. What is the main goal of K-Means clustering?

Solution and Explanation:
The main goal of K-Means clustering is to minimize the sum of squared distances between the data points and their assigned cluster centers.

Key Points:
  • K-Means is an iterative algorithm that finds K clusters.
  • The algorithm updates cluster centers to minimize distance to the points.




36. What does PCA (Principal Component Analysis) do?

Solution and Explanation:
PCA is a technique used to reduce the dimensionality of a dataset while retaining as much information as possible by preserving the most important features.

Key Points:
  • PCA helps with reducing the number of features in high-dimensional data.
  • It projects data into a lower-dimensional space while retaining the variance in the data.




37. What is cross-validation used for?

Solution and Explanation:
Cross-validation is used to assess the performance of a machine learning model by testing it on different subsets of the data, thus ensuring the model generalizes well.

Key Points:
  • It helps prevent overfitting by evaluating the model on different data subsets.
  • Common techniques include k-fold cross-validation.




38. What is the difference between accuracy and precision?

Solution and Explanation:
Accuracy measures the overall correctness of a model, while precision focuses on how well the model performs on positive predictions.

Key Points:
  • Accuracy: The proportion of correct predictions out of all predictions.
  • Precision: The proportion of true positives out of all positive predictions made.




39. Which metric is most useful when dealing with imbalanced datasets?

Solution and Explanation:
Precision and recall are more useful metrics for imbalanced datasets. Recall is especially important for ensuring the minority class is well predicted.

Key Points:
  • Accuracy can be misleading when the dataset is imbalanced.
  • Precision and recall provide more insight into model performance for imbalanced data.




40. What are hyperparameters in Machine Learning?

Solution and Explanation:
Hyperparameters are the settings or configurations set before the training process begins (e.g., learning rate, number of trees in a random forest).

Key Points:
  • They control the learning process and model structure.
  • Hyperparameters must be set manually or using optimization techniques.




41. What is the key difference between Grid Search and Random Search?

Solution and Explanation:
Grid Search tests all possible combinations of hyperparameters, while Random Search randomly selects combinations to find the optimal values.

Key Points:
  • Grid Search exhaustively searches the parameter space.
  • Random Search randomly explores the parameter space, which is often more efficient.




42. Why is hyperparameter tuning important?

Solution and Explanation:
Hyperparameter tuning is important to improve model performance, optimize the model's ability to generalize to new data, and prevent overfitting.

Key Points:
  • Hyperparameter tuning helps the model find optimal configurations.
  • It improves the model's predictive power and efficiency.




43. What is Time Series Data?

Solution and Explanation:
Time Series Data is a sequence of data points collected or recorded at time-ordered intervals.

Key Points:
  • Time series data involves time as an essential factor.
  • Common examples include stock prices, weather data, and economic indicators.

Key Points:

  • Clustering techniques like K-Means are used for grouping similar data points.




45. What does ARIMA stand for in time series analysis?

Solution and Explanation:
ARIMA stands for Auto-Regressive Integrated Moving Average, which is a popular model used for time series forecasting.

Key Points:
  • ARIMA combines autoregressive (AR) and moving average (MA) techniques.
  • The 'I' in ARIMA stands for 'Integrated', which refers to differencing the data to make it stationary.

55. What is Reinforcement Learning (RL)?

Solution and Explanation:
Reinforcement Learning is a type of machine learning where an agent learns to make decisions by interacting with an environment and receiving feedback in the form of rewards or penalties.

Key Points:
  • RL is used in applications like game playing, robotics, and recommendation systems.
  • The agent learns by trial and error, maximizing cumulative rewards over time.




56. In Reinforcement Learning, what is the role of the "reward function"?

Solution and Explanation:
The reward function determines the goal of the agent by assigning rewards for good actions and penalties for bad ones. It helps the agent learn to take actions that maximize long-term rewards.

Key Points:
  • The reward function is central to how an agent learns in Reinforcement Learning.
  • It helps guide the agent's behavior by providing feedback on its actions.




57. Which algorithm is commonly used in Reinforcement Learning?

Solution and Explanation:
Q-Learning is a model-free algorithm used in Reinforcement Learning. It allows agents to learn how to act optimally in an environment through trial and error.

Key Points:
  • Q-Learning is based on the concept of Q-values, representing the expected reward for a given state-action pair.
  • It is widely used in tasks like game playing and robotics.

Feature selection involves selecting the most relevant features for a model. This helps improve model performance and reduce complexity by removing irrelevant or redundant features.

Key Points:

Key Points:

60. Which method is used for feature selection?

Solution and Explanation:
Recursive Feature Elimination (RFE) is a method for feature selection that recursively removes the least important features based on model performance.

Key Points:
  • RFE helps reduce overfitting by eliminating irrelevant features.
  • It works by training models on different subsets of features and ranking them by importance.

Key Points:
  • Bias can arise from skewed data, unfair features, or flawed algorithms.
  • Mitigating bias is critical for creating fair and ethical AI systems.

Key Points:
  • Bias can also be mitigated by regular audits and fairness constraints.
  • Increasing model complexity or reducing dataset size may not always help reduce bias.

67. What is Transfer Learning?

Solution and Explanation:

Key Points:
  • It saves training time and resources.
  • It is especially useful when the new task has limited data.

69. What is an Autoencoder?

Solution and Explanation:
An Autoencoder is a type of neural network used for unsupervised learning and dimensionality reduction. It learns an efficient representation of the input data by encoding and then decoding it.

Key Points:
  • Autoencoders are typically used for feature extraction or noise reduction.
  • They consist of an encoder (compresses input) and a decoder (reconstructs input).




70. What is the purpose of an ROC curve in model evaluation?

Solution and Explanation:
The ROC (Receiver Operating Characteristic) curve is used to visualize the performance of a binary classification model by plotting the true positive rate (sensitivity) against the false positive rate (1 - specificity).

Key Points:
  • The area under the ROC curve (AUC) is used to evaluate the model's discriminative ability.
  • A model with a higher AUC is better at distinguishing between positive and negative classes.
