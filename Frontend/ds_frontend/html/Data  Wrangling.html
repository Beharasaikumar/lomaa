1. What is the primary purpose of using DataFrames in Pandas?

Correct Option: B) To handle and manipulate structured data efficiently

Solution and Explanation:
DataFrames are a two-dimensional labeled data structure in Pandas, designed to hold structured data like tables. They provide powerful and flexible tools to manipulate, filter, and analyze data.

**Key advantages of DataFrames**:
1. Label-based indexing for rows and columns.
2. Handling of missing data.
3. Easy data filtering and transformation.

Correct Option: C) By using pd.DataFrame() and passing the dictionary as an argument

Solution and Explanation:

**Example**:
`df = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})`

61. What is broadcasting in NumPy?

Correct Option: A) It allows NumPy to perform element-wise operations on arrays of different shapes.

Solution and Explanation:
Broadcasting refers to NumPys ability to perform arithmetic operations on arrays of different shapes by automatically expanding the smaller array to match the shape of the larger one.

**Example**:
`arr1 = np.array([1, 2, 3])`
`arr2 = np.array([10])`
`arr1 + arr2` will broadcast `arr2` to the shape of `arr1` and perform element-wise addition.




62. What will be the result of broadcasting a 2x3 array with a 1x3 array?

Correct Option: B) The 1x3 array will be broadcasted to a 2x3 array and then the element-wise operation will be performed.

Solution and Explanation:
When a 1x3 array is broadcasted with a 2x3 array, the 1x3 array will be expanded across the rows to match the shape of the 2x3 array. The operation will then be performed element-wise.

**Example**:
`arr1 = np.array([[1, 2, 3], [4, 5, 6]])`
`arr2 = np.array([1, 2, 3])`
`arr1 + arr2` will result in `[[2, 4, 6], [5, 7, 9]]`.




63. Which of the following is a necessary condition for broadcasting to work?

Correct Option: D) The size of the dimensions must be compatible.

Solution and Explanation:
For broadcasting to work, the dimensions of the two arrays must be compatible. This means that for each dimension, the size of one array must be either 1 or the same as the other array.

**Example**:
`arr1 = np.array([1, 2, 3])`
`arr2 = np.array([[10], [20]])`
These arrays can be broadcasted to perform element-wise operations.




64. What happens if broadcasting is not possible due to incompatible shapes?

Correct Option: C) An error will be raised.

Solution and Explanation:
If broadcasting is not possible due to incompatible shapes, NumPy will raise a `ValueError` indicating that the shapes are not aligned for broadcasting.

**Example**:

`arr2 = np.array([10, 20])`
This will raise a `ValueError` because the shapes are incompatible.




65. How does broadcasting handle arrays with dimensions of size 1?

Correct Option: B) The dimension of size 1 is stretched to match the larger array.

Solution and Explanation:

**Example**:
`arr1 = np.array([1, 2, 3])`
`arr2 = np.array([[10], [20]])`
The second array will be broadcasted across the rows of the first array to perform the operation element-wise.
