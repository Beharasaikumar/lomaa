EXPLORATORY DATA ANALYSIS

1. What is the primary purpose of data cleaning in data science?

Correct Option: B) Removing errors and inconsistencies from data

Solution and Explanation:
Data cleaning ensures the dataset is accurate, consistent, and ready for analysis by handling issues such as missing values, duplicate records, and outliers.

**Key Data Cleaning Tasks**:
1. Removing duplicates.
2. Handling missing values.
3. Correcting errors in the dataset.




2. Which method is commonly used to handle missing data in a dataset?

Correct Option: B) Imputation

Solution and Explanation:
Imputation involves filling in missing values with estimated ones based on available data, making the dataset complete for analysis.

**Methods to Handle Missing Data**:
1. Imputation.
2. Deletion.
3. Substitution with a constant.




3. Which of the following is a common data cleaning technique for handling outliers?

Correct Option: C) Capping or removing outliers

Solution and Explanation:
Outliers can skew the analysis. Capping (replacing with a threshold) or removing them helps in creating a more reliable dataset for analysis.

**Handling Outliers**:
1. Capping values.
2. Removing extreme values.
3. Transforming data.




4. What is the role of standardization in data cleaning?

Correct Option: B) To bring features to a similar scale

Solution and Explanation:
Standardization ensures that different features with different units are brought to a common scale, improving the performance of many machine learning algorithms.

**Benefits of Standardization**:
1. Equal scale for features.
2. Prevents dominance of certain features in modeling.
3. Improves algorithm convergence.




5. Which is a common issue that arises during the data cleaning process?

Correct Option: D) Balancing between data accuracy and completeness

Solution and Explanation:
Balancing the need for accurate data while ensuring that it is complete and ready for analysis is a common challenge during data cleaning.

**Challenges in Data Cleaning**:
1. Managing missing data.
2. Correcting errors.
3. Ensuring data consistency.




6. Which of the following methods is commonly used to handle missing data in categorical variables?

Correct Option: B) Imputation with the most frequent category

1. Imputation with the mode.
2. Imputation with a placeholder category.
3. Deleting rows with missing values.

11. Which of the following methods is commonly used for detecting outliers in a dataset?

Correct Option: A) Z-score method

Solution and Explanation:
The Z-score method detects outliers by calculating how many standard deviations a data point is from the mean. A Z-score greater than a threshold value (e.g., 3 or -3) is considered an outlier.

**Common Methods for Outlier Detection**:
1. Z-score.
2. IQR (Interquartile Range).
3. Visual methods (boxplots, scatter plots).




12. What does the IQR (Interquartile Range) method for outlier detection involve?

Correct Option: C) Identifying data points outside the range of Q1 - 1.5*IQR and Q3 + 1.5*IQR

Solution and Explanation:
The IQR method identifies outliers as data points that fall below the first quartile (Q1) minus 1.5 times the IQR or above the third quartile (Q3) plus 1.5 times the IQR.

**Outlier Detection with IQR**:
1. Compute the IQR (Q3 - Q1).
2. Define the lower and upper bounds as Q1 - 1.5*IQR and Q3 + 1.5*IQR.
3. Identify values outside this range as outliers.

Correct Option: D) Distorted model performance

Solution and Explanation:

**Consequences of Incorrect Outlier Handling**:
1. Biased results.
2. Decreased accuracy.
3. Overfitting.




14. Which of the following is an appropriate action when a dataset has a large number of extreme outliers?

Solution and Explanation:

**Actions for Extreme Outliers**:
1. Use robust models (e.g., decision trees).
2. Apply data transformations (e.g., log or square root transformations).
3. Consider removing or capping outliers if necessary.

19. What is the difference between Standardization and Normalization?

Correct Option: D) Standardization rescales data to have mean 0 and standard deviation 1, while normalization rescales data to a fixed range

Solution and Explanation:
Standardization transforms the data to have a mean of 0 and a standard deviation of 1, while normalization scales data to a predefined range, typically [0, 1].

**Difference between Standardization and Normalization**:
1. Standardization: mean = 0, std = 1.
2. Normalization: rescaling data to a specific range (e.g., [0, 1]).

21. What is the main goal of data normalization?

Correct Option: B) To rescale data to a fixed range, typically [0, 1]

**When to Use Normalization**:

22. What is the standard procedure to standardize data?

Correct Option: C) Subtract the mean and divide by the standard deviation

Solution and Explanation:
Standardization, also known as Z-score normalization, transforms data by subtracting the mean and dividing by the standard deviation, making it have a mean of 0 and a standard deviation of 1.

**When to Use Standardization**:
1. When data has a Gaussian distribution.
2. For algorithms that are sensitive to the scale of the data, such as linear regression, SVM, and PCA.

**Mitigating Issues with Outliers**:
1. Use robust scaling techniques.
2. Remove or handle outliers before normalization.

**Interpretation of p-value**:
1. p < 0.05: Reject the null hypothesis (variables are dependent).
